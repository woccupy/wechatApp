 This project is about wechat application architecture.
 Whth this code, you can easily build up your own wechat application.
 The whole architecture is consisted of message manager, state manager,
 and wechat components etc.

 If you want to build up your own wechat application. All you need to do is
follow the steps below .
 1, make a component.
 2, design your own status and messages.
 3, connect your component and message together.Don't forget letting your 
    status can be jumped by other statuses.
